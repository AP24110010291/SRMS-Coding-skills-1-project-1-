# Student Record Management System (SRMS)

This is a simple C programming project created as part of the **Coding Skills - 1** course at **SRM University, AP**.

The system helps an admin manage student academic records such as:
- Roll Number
- Student Name
- Marks

All data is stored permanently in a text file, which makes this system simple and easy to use.
